So i was working on PostgreSQL and collected few commands for assistance in future. This is basic commands that can help MySQL user to get started.

PostgreSQL is accessible by only postgres user (by default) so you have to use sudo with every command executing from root user.

 

Entering PSQL prompt/Command line : sudo -u postgres psql

 

Get a list of databases in Postgres:
From shell prompt : sudo -u postgres psql -l
From PSQL prompt : \l [Note: This is backslash followed by lower-case L]

 

Create a Database in PostgreSQL:
From shell prompt : sudo -u postgres createdb testdb
From PSQL prompt : CREATE DATABASE testdb

 

Delete/Drop a Database in PostgreSQL:
From shell prompt : sudo -u postgres dropdb testdb
From PSQL prompt : DROP DATABASE testdb

 

Backup/Dump a Database in PostgreSQL:
From shell prompt : sudo -u postgres pg_dump testdb > backup.out

 

Restore a Database in PostgreSQL:
From Shell Prompt : sudo -u postgres psql -f backup.out testdb

 

Other usefull command from PSQL prompt:
Selecting Database : \c testdb
Getting List of tables : \d OR \dt
Exiting PSQL prompt : \q
Getting Help : \? OR \h OR \h CREATE

 

Hope this will help.
